# HH API System

Анализатор вакансий с HeadHunter API на Python.

## Описание

Скрипт анализирует вакансии по заданному запросу, извлекает ключевые навыки, рассчитывает средние зарплаты с конвертацией валют, и сохраняет результаты в JSON файл.

## Функции

- Поиск вакансий по ключевому слову
- Извлечение навыков из описаний вакансий
- Конвертация зарплат в рубли
- Расчет статистики по навыкам и зарплатам
- Сохранение результатов в result.json

## Установка

1. Клонируйте репозиторий:
   ```
   git clone https://github.com/todmiv/hh_api_system.git
   cd hh_api_system
   ```

2. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

## Запуск

```
python main.py
```

Введите интересующую вакансию (например, "python").

## Зависимости

- requests
- pycbrf

## Требования

- Python 3.6+
- Интернет-соединение для доступа к HH API
